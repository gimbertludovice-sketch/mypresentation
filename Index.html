Activity 1 — Basic CSS Styles

Making Elements look the way you want

Learning objective

Apply CSS properties directly to HTML element selectors — targeting every <h1> or every <p> on the page at once.

Requirements met

h1 → color: blue, centered
p → font-size: 18px, color: gray

Activity 1 — Code & Live Result

/* Element selectors — apply to ALL matching tags */
h1 {
  color: blue;
  text-align: center;
  font-size: 2rem;
}

p {
  font-size: 18px;
  color: gray;
}
↓ Rendered output

Understanding CSS Fundamentals

CSS controls the visual presentation of HTML elements. With just a few properties, you can transform how a page looks and feels.

Activity 2 — Class & ID Selectors

Targeting with precision

ID selector · #title

Targets one unique element. Highest specificity. Used for the page heading: green text, centered alignment.

Class selector · .info

Reusable across many elements. Applied to paragraph blocks: beige background, 16px font, comfortable padding.

Activity 2 — Code & Live Result

/* ID selector — unique element only */
#title {
  color: green;
  text-align: center;
}

/* Class selector — reusable on many elements */
.info {
  background-color: beige;
  font-size: 16px;
  padding: 20px 24px;
}
↓ Rendered output

CSS Selectors in Action

An ID selector (#title) targets one unique element and applies the highest specificity styles.

A class selector (.info) applies the same beige block style to every paragraph that carries it.

Selector Reference

Activity 1 vs Activity 2 — key differences

Selector type Syntax used Target Style applied Activity
Element h1, p Every tag of that type Blue heading · gray 18px text Act 1
ID #title One unique element Green · centered Act 2
Class .info Any element with that class Beige · 16px · padded Act 2
